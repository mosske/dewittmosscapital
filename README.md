# DeWitt Moss Capital Website

A modern, professional website for DeWitt Moss Capital - Premier Specialists in Capital-Raising Solutions.

## Features

- 📱 Fully responsive design
- 🎨 Professional color scheme (White, Dark Gold, Black, Turquoise)
- ⚡ Built with React, TypeScript, and Tailwind CSS
- 🚀 Fast performance with Vite
- 🎯 Three main pages: Home, About, Contact
- 📧 Contact form
- 🌍 International market focus display

## Tech Stack

- React 18
- TypeScript
- Tailwind CSS
- Vite
- React Router

## Installation

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
