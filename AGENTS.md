# Project Specific Rules & Directives

## Image Handling Directive
- **Strict Image Integrity**: When the user uploads or adds an image file to include in the site, use the image file exactly as provided verbatim.
- **No Replacement or Edits**: Do NOT replace provided images with generated or AI images, do NOT amend, edit, crop, transform, or alter the image in any way, and do NOT use discretion to make judgments or edits to the image.
- **Literal Execution**: Follow user instructions for image files strictly with zero modifications or overthinking.
